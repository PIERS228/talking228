GitHub 上传指南

1

登录 GitHub

访问 github.com 并登录您的账户

2

进入您的仓库

在右上角点击您的头像，选择 "Your repositories"

点击您新创建的仓库名称进入仓库主页

3

上传文件

在仓库主页，找到并点击 "Add file" 按钮

从下拉菜单中选择 "Upload files"

将保存的 cat-therapist.html 文件拖放到指定区域

4

提交更改

滚动到页面底部，在 "Commit changes" 区域：

  • 填写简短描述，例如："添加猫猫心理师网页"
  • 点击绿色 "Commit changes" 按钮
5

启用 GitHub Pages

转到仓库的 "Settings" 选项卡

在左侧菜单选择 "Pages"

在 "Source" 部分：

  • 分支选择 "main"
  • 文件夹选择 "/ (root)"
  • 点击 "Save"
6

访问您的网站

等待几分钟让 GitHub 构建您的网站

返回 "Settings > Pages"，您会看到类似这样的链接：

https://<您的用户名>.github.io/<仓库名>

点击链接即可访问您的猫猫心理师网站！
